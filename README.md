# AI CLI Hub

A central hub for your AI CLI tools.

## Features

- **Gemini CLI**
- **GitHub Copilot CLI**
- **Mistral Vibe**
- **Claude Code**

## Installation

To register the `ai` command globally on your system:

1.  Open your terminal in this directory (`ai-hub`).
2.  Run the following command:

    ```bash
    npm link
    ```
    
    *Or*

    ```bash
    npm install -g .
    ```

3.  Now you can run the tool from anywhere by typing:

    ```bash
    ai
    ```

## Usage

Run `ai` and use the arrow keys to select the tool you want to launch.

## Requirements

- Node.js installed.
- The respective CLI tools (e.g., `gemini`, `gh`, `mistral`, `claude`) must be installed and available in your system PATH for them to launch successfully.
